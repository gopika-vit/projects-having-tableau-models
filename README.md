# Tableau Dashboard Project

## Overview

This project uses Tableau to analyze, visualize, and share data through interactive dashboards and reports.

Tableau enables users to connect to multiple data sources, transform data, create visualizations, and publish insights for business intelligence and analytics.

## Features

- Interactive dashboards
- Data visualization and storytelling
- Real-time analytics
- Multiple data source connectivity
- Custom charts and KPI tracking
- Dashboard sharing and collaboration

## Prerequisites

Before opening this project, ensure you have:

- Tableau Desktop installed
- Access to the required data sources
- Appropriate permissions for Tableau Server or Tableau Cloud (if publishing)

## Installation

1. Install Tableau Desktop.
2. Download or clone this repository.
3. Open the `.twb` or `.twbx` file in Tableau Desktop.

## Usage

### Open Dashboard

1. Launch Tableau Desktop.
2. Open the project workbook (`.twb` or `.twbx`).
3. Refresh data if required.
4. Explore the available dashboards and worksheets.

### Publish Dashboard

1. Click **Server → Publish Workbook**.
2. Select your Tableau Server or Tableau Cloud destination.
3. Configure permissions and sharing settings.
4. Publish the dashboard.

## Best Practices

- Keep dashboards simple and focused.
- Use consistent colors and formatting.
- Limit the number of filters when possible.
- Optimize data extracts for better performance.
- Add descriptive titles and tooltips.

## Troubleshooting

### Data Not Refreshing

- Verify data source connectivity.
- Check database credentials.
- Confirm file paths are correct.

### Slow Dashboard Performance

- Reduce the number of worksheets.
- Use extracts instead of live connections.
- Remove unused fields and calculations.
